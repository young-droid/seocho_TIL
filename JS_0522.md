cpu 에게 부하가 걸리는 코드는 쓰지 말자. 
ex) 
for (let i = 0; i < fn(); i++) {...}
    -> fn()을 n 번씩 루프돌리는 짓. 바꾸면: 

const limit = fn();
for (let i = 0; i < limit; i++)
    -> 이러면 맨 처음 limit 에서 fn을 한번만 불러오고 끝, 훨씬 빠르다
